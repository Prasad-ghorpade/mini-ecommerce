# 🛒 Mini eCommerce App (Next.js)

A modern mini eCommerce application built using Next.js (App Router) with clean UI, reusable components, and basic cart + authentication features.

---

## 🚀 Features

- 🔐 Authentication (Login / Logout)
- 🛍️ Product listing (API based)
- 📦 Product details page
- 🛒 Cart functionality (Add / Remove / Quantity)
- 💰 INR currency conversion
- 👤 User session handling (localStorage)
- 🎨 Clean and responsive UI
- ⚡ Reusable component-based architecture

---

## 🧠 Tech Stack

- Next.js (App Router)
- React
- Context API (State Management)
- Tailwind CSS
- FakeStore API

---
